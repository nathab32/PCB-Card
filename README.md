# PCB Card

![image of PCB](https://raw.githubusercontent.com/nathab32/PCB-Card/3c2ca3ead15cd8f8822be567be30546e786c707a/Screenshot%202026-07-02%20055419.png)
This is a PCB Business Card I made for Fallout! It has a simple circuit powered by a CR2032 Button Cell, and various silkscreen graphics. The circuit just blinks two LEDS on and off.
